# AI Alpaca: Autonomous Trading Bot

<Lead>An autonomous trading bot powered by Claude Code and the **official Alpaca MCP server**. Executes trades based on market data, news, and social signals — fully autonomously.</Lead>

## Overview

AI Alpaca uses Claude Code with MCP (Model Context Protocol) to connect directly to Alpaca's trading API. The bot runs on a cron schedule, gathering market intelligence and making trading decisions without human intervention.

### Key Features

- **Official Alpaca MCP Server** — Uses `alpacahq/alpaca-mcp-server` for direct API access to stocks, ETFs, options, and crypto
- **Autonomous Decision-Making** — Claude Code analyzes market data, news, and social signals to execute trades
- **Self-Modifying Strategy** — The bot maintains and updates its own trading plan and strategy documents
- **Dockerized** — Runs as a containerized service with persistent state and logging
- **Flexible Scheduling** — Trades every 15 minutes during market hours, with analysis-only ticks off-hours

### How It Works

1. Load current portfolio state and trading strategy
2. Gather market data via the Alpaca MCP server (quotes, bars, positions)
3. Assemble context and invoke Claude Code for analysis
4. Parse trading decisions and execute orders
5. Log all actions and update state for the next tick

# React `is-online` Context and Hook

> A React context provider for accurately determining if a browser is online or not.

Works with Electron.

This component was created for use with [Cinematic](https://github.com/lacymorrow/cinematic).

## Installation

```sh npm2yarn
npm install react-is-online-context --save
```

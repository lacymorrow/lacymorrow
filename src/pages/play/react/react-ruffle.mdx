   # React Ruffle: Flash Media Player Component

A React component for rendering Flash & ActionScript content using the Rust-based Ruffle emulator.

This component was created for use on this website.

## Demo

> This demo is using the `react-ruffle` component to render a Flash swf file. Credit to [Trevor Van Meter](https://www.heytvm.com/work) for the original Flash game [_Flyguy_](https://en.wikipedia.org/wiki/Fly_Guy_(video_game))

## Installation

```sh npm2yarn
npm install react-ruffle --save
```
